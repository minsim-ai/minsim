# Segment Reaction Radar

## Decision

지역·성별 결과를 접힌 부록 표가 아니라 결과 화면의 1급 의사결정 영역인 `세그먼트 반응 레이더`로 제공한다.

- `creative_testing`의 A/B/C/D 선택지와 `churn_prediction`의 유지/관망/이탈을 별도 의미 체계로 렌더링한다.
- 이탈 예측 지도는 대표 반응의 의미 색상과 이탈률 강도를 함께 사용한다.
- 높은 비율과 높은 신뢰도를 같은 의미로 취급하지 않는다. 표본 수와 신뢰도는 항상 수치 옆에 표시한다.
- 기본 지역 정렬은 최고 비율이 아니라 신뢰 우선이다.
- 지역 지도는 키보드로 선택할 수 있고 동일 데이터를 순위 목록으로도 제공한다.
- 결과의 `country_id`에 따라 설치된 10개국의 행정경계 지도를 선택한다. 한국만 지도를 표시하고 다른 국가를 목록으로 축소하지 않는다.
- 국가별 원본 지역 키는 state code, 현지어 지명, 악센트 차이를 정규화한 뒤 경계 path와 연결한다.
- 하나의 경계에는 여러 원본 지역 키를 연결할 수 있다. 인도는 영어·로마자 힌디어·데바나가리 78개 key를 35개 경계에, 벨기에는 네 언어 12개 key를 3개 경계에 연결한다.
- ISO형 경계 ID에만 국가 코드 suffix 매칭을 허용한다. 프랑스 지명형 ID의 하이픈 조각을 다른 département로 오인하지 않도록 본토 96개 key parity를 검증한다.
- 지도 경계 출처와 라이선스를 결과 범례와 정적 attribution 문서에 남긴다.
- 성별 표본이 한쪽에만 존재하면 누락 집단을 숨기지 않고 비교 불가 상태를 명시한다.

## Semantic Contract

### Choice comparison

- 표시: `A안`, `B안`, `C안`, `D안`
- 초점 지표: 전체 1위안의 지역별 선택률

### Churn prediction

- 표시: `유지`, `관망`, `이탈`
- 금지 표현: `유지안`, `관망안`, `이탈안`, `유안`, `관안`, `이안`
- 초점 지표: 지역별 이탈률
- 의미 색상: 유지=청록, 관망=호박색, 이탈=코랄/적색

### Market segmentation

- 표시: free-text 세그먼트 라벨 그대로 (예: `감성선물 찾는 직장인`)
- 금지 표현: 세그먼트 라벨에 `안` 접미사 붙이기, 초점 라벨 `N/A`
- 결과 어댑터 모드: `segment` (`metrics.segment_counts` / `segment_pct`)
- 초점 지표: `recommended_first_target` 또는 1위 세그먼트의 지역별 점유율
- KPI 라벨: `{focusLabel} 점유율`
- 표시 열: 상위 8개 세그먼트 + 나머지 `기타` 합산 (연령/지역/성별 breakdown 동일 폴딩)
- 연령 분포 표와 시장 반응 막대는 위 열을 공유한다

### Reliability

- 50명 이상: 높음
- 30~49명: 보통
- 10~29명: 낮음
- 10명 미만: 참고

지도 색만으로 의미를 전달하지 않고 대표 반응, 비율, 표본 수, 신뢰도 텍스트를 함께 제공한다.

## Result Layout

1. 요약 다음에 `세그먼트` 앵커를 노출한다.
2. KPI 스트립에 전체 초점 지표, 신뢰 가능한 주의 지역, 관측 최고 지역, 성별 표본 범위를 표시한다.
3. 데스크톱은 지도 2/3 + 정렬 가능한 지역 목록 1/3, 모바일은 한 열로 재배치한다.
4. 지역 선택 시 전체 대비 차이, 표본 신뢰도, 해석, 후속 질문 이동을 표시한다.
5. 지도 아래에 성별 분포를 별도 카드로 제공하며 교차 분석으로 오인시키지 않는다.

### Multi-country boundary levels

- ADM1: 미국, 일본, 인도, 브라질, 베트남, 엘살바도르, 벨기에.
- ADM2: 프랑스 département, 싱가포르 planning area.
- 대한민국은 기존 시도 SVG를 유지한다.
- 데이터에 존재하지만 경계 원본의 행정구역 개편 시점과 맞지 않는 지역은 순위 목록에 유지하고 지도에서는 중립 경계로 표시한다.
- 현재 경계 원본 범위 밖인 미국 Puerto Rico와 프랑스 해외 département는 순위 목록에 유지한다. 해당 경계가 포함된 검증된 원본으로 교체하기 전에는 다른 본토 경계에 잘못 연결하지 않는다.

## Landing Preview

공개 홈페이지에는 사용자 결과가 아닌 고정 예시 데이터를 사용한다.

- `예시 데이터` 라벨을 항상 노출한다.
- 미니 지도, 핵심 지표 3개, 상위 지역 목록만 보여준다.
- CTA는 프로젝트 시작 흐름으로 연결한다.
- 실제 시장 성과가 아니라 합성 패널의 `세그먼트 반응`임을 명시한다.

## Data Boundary

현재 API는 지역×성별 교차표가 아니라 각각의 주변 분포만 제공한다. 따라서 UI는 성별 탭으로 지역 지도를 필터링하는 것처럼 보이게 만들지 않는다. 실제 교차 필터는 백엔드 결과 계약이 추가될 때 별도 버전으로 확장한다.
