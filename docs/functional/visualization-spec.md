# 결과 시각화 표준 — 노드 그래프 + 군중감

> **이 문서는 KoreaSim의 핵심 UX 차별 포인트를 정의한다.**
> Aaru는 통계 표·차트 위주의 "B2B 컨설팅 리포트" 스타일.
> KoreaSim은 **"수많은 사람이 직접 말하는 것처럼 느끼는"** 인터랙티브 시각화로 차별화한다.

---

## Goal (UX 의도)

1. 시뮬레이션 결과를 **숫자**가 아니라 **사람**으로 느끼게 한다
2. 200~1,000명 시뮬레이션의 **군중감**을 시각적으로 전달
3. 깊이 있는 인사이트는 **드릴다운**으로 점진적으로 노출
4. 수천명 규모에서도 **브라우저가 매끄럽게** 동작 (LOD 최적화)

---

## 1. 노드 그래프 (안 비교 시각화)

### 적용 시뮬레이션
- [[01-creative-testing|크리에이티브 테스트]]
- [[04-value-proposition|가치 제안 테스트]]
- [[06-competitive-positioning|경쟁적 포지셔닝]]

### 시각화 구조

```
┌──────────────────────────────────────────────┐
│  [A 안]                  [B 안]              │
│   ⬤ 87명               ⬤ 64명              │
│   ╱ ╲                   ╱                   │
│  ╱   ╲                 ╱                    │
│ 30대   40대          전 연령                 │
│ 직장인  주부                                 │
│                                              │
│             [C 안]                           │
│              ⬤ 42명                         │
└──────────────────────────────────────────────┘
```

### 노드 정의
- **메인 노드**: 각 안 (창의·제안·경쟁사) — 크기 = 선택 수
- **세그먼트 노드** (자식): 해당 안을 선택한 주요 세그먼트 (연령대·직업·지역)
- **연결선**: 세그먼트가 둘 이상 안을 비슷하게 선호 → 회색 점선

### 인터랙션
- **호버**: 해당 노드의 핵심 지표 툴팁
- **클릭**: 우측 패널에 세부 분포 + 상위 응답 이유 + 페르소나 카드
- **드래그**: 노드 재배치 (force-directed)
- **줌/팬**: 마우스 휠 + 드래그

### 기술 스택
- **D3.js force-directed graph** 또는 **react-flow**
- 또는 React 기반 chart component
- Streamlit fallback은 외부 데모 기준 구현 대상이 아님

## 2. 사람 아이콘 + 말풍선 (대표 인터뷰)

### 적용 시뮬레이션
- 모든 시뮬레이션 (공통 컴포넌트)

### 시각화 구조

```
┌──────────────────────────────────────────────┐
│ [A 안] 87명 선택                             │
│                                              │
│  👤  👤  👤  👤  👤  👤  👤  👤  👤  👤   │
│ "신선해서        "출근길에                   │
│  좋아"           눈에 띌듯"                  │
│                                              │
│  👤  👤  👤  👤  👤  👤  👤  👤  👤  👤   │
│         "내 일상에                           │
│          어울려"                             │
│                                              │
│  ... (10x10 grid = 100명 표시)              │
└──────────────────────────────────────────────┘
```

### 동작
- 안별로 사람 아이콘 그리드 (10x10 = 100명, 그 이상은 +N 표시)
- 색상 구분:
  - 🟢 강한 선호 (높은 설득력 점수)
  - 🟡 중립
  - 🔴 부정 (다른 안 선호)
- **말풍선 자동 순환**: 3~5초마다 다른 페르소나의 대표 발언 노출
- 발언 길이: 5~12자 (기본) 또는 1문장
- 클릭 시: 해당 페르소나 전체 응답 + 인구통계 카드 (모달)

### 대표 발언 선정 로직
```python
def select_representative_quotes(reasons: list[str], n: int = 12) -> list[str]:
    """
    1. 빈도 기준 상위 키워드 추출
    2. 각 키워드를 포함한 짧은 발언 1개씩 샘플링
    3. 너무 짧거나 긴 응답 제외 (5~80자)
    4. 다양성 확보 (같은 표현 중복 제거)
    """
    ...
```

### 군중감 효과
- 처음 로드 시 사람 아이콘이 **하나씩 fade-in** (1초 안에 100명 채워짐)
- 시뮬레이션 진행 중 실시간 추가 (응답 들어올 때마다 아이콘 1개)
- "당신이 한국 인구 한가운데 서 있는 듯한" 느낌

## 3. 군중감 성능 설계 (LOD: Level of Detail)

### 문제
- 1,000명 페르소나를 모두 DOM 노드로 렌더링하면 브라우저 멈춤
- 수천 개 SVG·canvas 객체 = 성능 저하

### 해결: LOD (Level of Detail)

| 줌 레벨 | 표시 방식 | 노드 수 |
|--------|---------|--------|
| 전체 (overview) | 클러스터 노드만 (그룹별 1개) | ~50~200 |
| 줌 인 1단계 | 클러스터 펼침 (대표 페르소나) | ~500 |
| 줌 인 2단계 (특정 영역) | 개별 페르소나 | ~100 in 영역 |

### 배경 표현 (대규모 시각화)
- 줌 아웃 시 페르소나는 **파티클 애니메이션** (발광 점)
- 또는 **실루엣 군중** (배경 일러스트)
- 줌 인 시에만 개별 형태로 변환

### 진행 중 애니메이션
- 시뮬레이션 응답이 들어올 때마다:
  - 진행률 바 갱신
  - 군중에 사람 아이콘 1개 추가 (fade-in)
  - 점차 그래프 채워지는 효과

### 기술 스택
- **Canvas API** (수천 개 점 렌더링용, SVG보다 10x 빠름)
- **deck.gl** (대규모 데이터 시각화 가능)
- 또는 단순화: **Plotly scatter with marker.size + opacity**

## 6. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **숫자보다 사람** | "67%" 대신 "67명이 이렇게 말했어요" |
| **점진적 노출** | overview → 세그먼트 → 페르소나 (3단계 드릴다운) |
| **한국어 자연스러움** | 말풍선 폰트는 본명조·맑은고딕, 영문 폰트 X |
| **부드러운 애니메이션** | 갑작스러운 등장 X, 항상 fade/slide 트랜지션 |
| **모바일 X** | 데스크톱 전용 (Phase 5에서 모바일 검토) |
| **색맹 친화** | 색뿐만 아니라 형태로도 구분 (✓, ✗, ?) |

## 7. 데모 임팩트 (왜 이게 중요한가)

### Aaru 데모 (예상)
> "이 광고는 47%의 선호도를 보였습니다. 30~40대 직장인에서 강세를 보였습니다."
> [숫자 + 막대 차트]

### KoreaSim 데모 (목표)
> [화면에 100명의 사람 아이콘이 점점 채워짐]
> [말풍선이 차례로 떠오름]
> "신선해서 좋아요" — 35세 여성, 서울 강남구
> "내 일상이랑 어울려" — 42세 남성, 경기 분당
> "출근길에 눈에 띌듯" — 28세 여성, 부산 해운대
>
> [클릭 시 모달] 박미영, 35세, 서울 강남구, 마케팅 매니저
> "신선한 과일을 보면 아침이 개운하게 시작될 것 같아요. 저는 평소에..."

→ **"진짜 사람들이 답한 것 같다"는 감정적 임팩트**가 의사결정자에게 강하게 작동
