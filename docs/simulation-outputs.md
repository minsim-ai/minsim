# 시뮬레이션 출력 스펙

> 9개 시뮬레이션 타입별 출력 형태 5종 정의. ResultsPage 개발 기준.

---

## 1. 크리에이티브 비교

| # | 출력 형태 | 예시 |
|---|---------|-----|
| 1 | **선호도 도넛 + 안별 대표 발언** | A안 49% "신선해서 좋아요" / B안 35% "직설적이라 좋네" |
| 2 | **사람 아이콘 군중 100명** | 안별로 fade-in, 클릭하면 인구통계 + 전체 응답 |
| 3 | **세그먼트별 1순위 매트릭스** | 20대 여성 강남→A / 50대 남성 부산→B / 30대 워킹맘→C |
| 4 | **안 간 키워드 유사도 그래프** | A·B 모두 "출근길" 언급 18회 → 회색 연결선 |
| 5 | **한국 문화 키워드 자동 추출** | "정" 32회 / "체면" 8회 / "가성비" 41회 / "프리미엄" 22회 |

## 2. 가격 최적화

| # | 출력 형태 | 예시 |
|---|---------|-----|
| 1 | **가격-수요 곡선 + 수익 곡선** | 4,500→78% / 5,500→55% / 6,500→32% / 수익 정점 5,500원 |
| 2 | **가격 탄력성 E + 해석** | E=1.8 (탄력적) → "10% 인상 시 매출 8% 감소 예상" |
| 3 | **세그먼트별 WTP 분포** | 강남구 평균 6,800원 / 노원구 5,100원 / 서울 vs 지방 격차 |
| 4 | **임계 거부 가격 분포** | "7,000원 넘으면 안 사" — 30대 워킹맘에서 47% |
| 5 | **가성비 vs 프리미엄 페르소나 비교** | 두 그룹 카드 + 대표 발언 + "가심비" 키워드 자동 |

## 3. 신제품 반응

| # | 출력 형태 | 예시 |
|---|---------|-----|
| 1 | **점수 분포 히스토그램 (1~5)** | 평균 3.7, 5점 28% / 1~2점 18% — 양극화 감지 |
| 2 | **"꼭 사겠다" 5점 페르소나 5명 카드** | 인구통계 + 전체 응답 + 구매 이유 → 마케팅 카피 힌트 |
| 3 | **거부 사유 자동 클러스터링** | "비싸다" 22건 / "기존 제품 만족" 18건 / "디자인" 12건 |
| 4 | **지역×연령 잠재 매출 매트릭스** | 30대 서울 60% × 인구 비중 = 매출 기여 1순위 |
| 5 | **취미·라이프스타일 매칭** | "위생 민감" 페르소나 점수 평균 4.5 vs 전체 3.7 |

## 4. 가치 제안

| # | 출력 형태 | 예시 |
|---|---------|-----|
| 1 | **VP 설득력 랭킹 (5점)** | "가장 빠른" 4.2 / "한국어 최적화" 3.8 / "가격" 3.1 |
| 2 | **직업군별 선호 VP 히트맵** | 개발자=빠름 / 임원=안전 / 자영업=가격 |
| 3 | **VP별 페르소나 인터뷰 5명** | "빠른" 선택자 5명 카드 (왜 빠른 게 중요한지) |
| 4 | **선택 vs 거부 키워드 양극 차트** | 선택 "신뢰감" / 거부 "재미없음" |
| 5 | **타겟 세그먼트 매칭률** | "30대 마케팅팀" 타겟 시 VP1 → 78% 적중, VP3 → 32% |

## 5. 시장 세분화

| # | 출력 형태 | 예시 |
|---|---------|-----|
| 1 | **6개 자동 세그먼트 + 라벨** | "건강 추구 중장년" 22% / "가성비 1인 가구" 18% / ... |
| 2 | **세그먼트별 대표 페르소나 카드 5명** | 세그먼트당 5명 (말풍선으로 핵심 니즈) |
| 3 | **세그먼트 거리 맵 (2D)** | 비슷한 세그먼트 가까이, 빈 공간 = 기회 |
| 4 | **세그먼트별 잠재 시장 규모** | "건강 추구 중장년" = 한국 인구 추정 270만 명 |
| 5 | **세그먼트별 추천 메시지·채널** | "프리미엄 워킹맘 → 인스타그램 + '시간 절약' 메시지" |
